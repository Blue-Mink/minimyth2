MiniMyth2
Overview

Minimyth2 is a small Linux distribution that turns a diskless computer into a MythTV frontend.

This version of MiniMyth2 is version @MM_VERSION_MINIMYTH@ and contains:

  • MythTV build from sources through @MM_VERSION_MINIMYTH@ with

    MythTV Patches:

    • Applied ticket9633 Progressbar on OSD
    • Applied ticket10710 Instant playback from playlist editor

    MythTV Plugins:

    • MythBrowser
    • MythGallery
    • MythGame(not integrated with Minimyth configuration)
    • MythMusic
    • MythNetvision
    • MythNews
    • MythWeather
    • MythZoneMinder

  • Major System Components:

    • Linux @LINUX_FULL_VERSION@
    • Xorg 7.7 with Xserver 1.20.8
    • LIRC 0.10.1
    • ALSA 1.2.2
    • MESA 20.2.1
    • Qt 5.15.1
    • Qt WebKit 5.212 gac8ebc6c

  • Major Build Tools/Libraries:

    • GNU Binutils 2.35
    • GCC compiler 10.2.0
    • LLVM compiler 9.0.1
    • GNU C Library 2.32

  • Key Video Drivers:

    • NVIDIA Xorg graphics driver @MM_NVIDIA_VERSION@
    • NVIDIA Xorg legacy graphics driver @MM_NVIDIA_LEGACY_VERSION@
    • Intel Xorg graphics driver git 20200131-55ced7c
    • ATI Xorg graphics driver 19.1.0
    • VMware Xorg graphics driver git 20200131-be1b588
    • Intel VAAPI driver 2.6.1
    • VAAPI libva library 2.3.0
    • DRM libdrm library 2.4.99
