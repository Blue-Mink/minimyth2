MiniMyth2
Overview

Minimyth2 is a small Linux distribution that turns a diskless computer into a MythTV frontend.

This version of MiniMyth2 is version @MM_VERSION_MINIMYTH@ and contains:

  • MiniMyth2 offers following key functions:
    • Support for zero-touch-provisioning. With proper DHCP/TFTP/HTTP servers configuration, adding new fully functional frontend requires only enabling PXE boot in frontend's BIOS
    • Universal kernel and infrastructure allowing to PXE boot of MiniMyth2 on BIOS and EFI systems
    • Auto-detection and auto-configuration for Nvidia/AMD VDPAU, Intel VAAPI and VMware with 1100+ supported graphic cards
    • Auto-detection and auto-configuration for IR Remote Receivers supported by Linux kernel with hot-pluging
    • Build-in support for external user equipment control with help of LIRC irblaster and simple command scripts language
    • Build-in OSD notifications for New Mails, Recording status, FE/BE hardware status, Upcoming recordings, etc
    • Initial configuration-less mode where system where new system can be booted and fully functional without any configuration
    • Full suspend-to-RAM based sleep/resume allowing fast (few sec) & reliable ON/OFF operation with appliance like experience
    • Netfilx/Showmax playback via Google Chrome/Mozilla FireFox with integrated IR remote control
    • BluRay backup/rip (based on MakeMKV; only OSS part) for BluRay playback/RIP/Copy to server storage
    • Thelephony VoIP SIP client allowing to make/receive/voice-mail SIP calls from MythTV GUI
    • Enhanced IR remote buttons function for supporting rescue system actions like mythfrontend/xserver restart
